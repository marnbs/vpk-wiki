{{< hint warning >}}
Ожоги являются одними из самых частых неотложных состояний. Тяжелые ожоги всегда требуют медицинской помощи.
{{< /hint >}}

## Основные виды ожогов
- термические  
- химические  
- электрические  

## Признаки тяжелого ожога
- обширная площадь  
- ожоги лица, дыхательных путей, паховой области  
- признаки шока  
- обугливание ткани  

## Первая помощь при термических ожогах

{{% steps %}}
### 1. Устранить источник жара
Потушить пламя, удалить горячий предмет.

### 2. Начать охлаждение
Охлаждать холодной водой 10-20 минут.

{{< image src="/images/first-aid/conditions/burns-cool-01.png" caption="Охлаждение ожога водой" >}}

### 3. Наложить стерильную сухую повязку
Не использовать жиры и кремы.

### 4. Дать обезболивание при наличии
Только если разрешено инструкциями.

### 5. При необходимости вызвать скорую
При поражении большой площади.
{{% /steps %}}

## Химические ожоги
- промывать большим количеством воды  
- при ожогах порошками — сначала удалить сухое вещество чистой тканью  

## Ожоги электричеством
- отключить источник тока  
- контролировать дыхание  
- обработать входные и выходные ранки
