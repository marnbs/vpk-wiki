Список основан на «Приложении 1» [Минздрава России](https://сдк.впккомбат.рф/media/minzdrav.pdf), стр 63.

## Основные категории изделий
- средства для временной остановки кровотечения  
- перевязочные материалы  
- изделия для СЛР  
- вспомогательные средства  

{{< hint info >}}
В различных организациях состав аптечек может дополняться, но минимальные требования соответствуют перечню Минздрава.
{{< /hint >}}

## Перевязочные материалы
- бинты марлевые разных размеров  
- стерильные салфетки  
- бактерицидные пластыри  
- рулонный лейкопластырь  

## Средства остановки кровотечения
- жгут кровоостанавливающий  
- перевязочный пакет  

## Средства для СЛР
- маска медицинская  
- перчатки  
- салфетки спиртовые  

## Прочие изделия
- ножницы  
- спасательное изотермическое покрывало  
- дополнительные материалы организации
