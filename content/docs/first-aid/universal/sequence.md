## 1. Оценка обстановки
- Определить угрозы для спасателя.
- Определить угрозы для пострадавшего.
- По возможности устранить опасность.

## 2. Проверка сознания
Слегка потрясти за плечо, громко спросить: «Что с вами?»

## 3. Проходимость дыхательных путей
- Закинуть голову.
- Поднять подбородок.
- При необходимости — выдвинуть нижнюю челюсть.

## 4. Оценка дыхания (10 сек)
Слушать → Смотреть → Ощущать.

## 5. Вызов скорой помощи
112 / 103 / 03.

## 6. Если дыхания НЕТ — начать СЛР  
Если дыхание ЕСТЬ — придать боковое устойчивое положение.

## 7. Осмотр пострадавшего (от головы до ног)
Голова → шея → грудь → спина → живот → таз → конечности.
